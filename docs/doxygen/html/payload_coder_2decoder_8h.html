Detailed Description

Payload decoder for muskrat trap LoRaWAN node. Decodes compact binary payloads into sensor and status fields.

Fields decoded: id, version, door status, catch detect, trap displacement, battery status, unix time. Usage: Call decodePayload() with buffer, then use getters to access fields.
