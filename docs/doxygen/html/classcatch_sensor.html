◆ setCatchStatus()

void catchSensor::setCatchStatus ( bool catchStatus)

Sets the catch status and updates the corresponding LED. This method updates the internal catch status of the sensor. Based on the new status, it controls rightRedLED on the HAN IoT Shield:

  • If a catch is detected (catchStatus is true), rightRedLED is turned ON.
  • If no catch is detected (catchStatus is false), rightRedLED is turned OFF.
    Parameters
    catchStatusBoolean value representing the new catch status. True indicates a catch, false indicates no catch.
